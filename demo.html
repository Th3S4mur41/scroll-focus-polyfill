Scroll Focus Polyfill Demo

Try it: Press Tab to navigate to the code block below. In browsers that need the polyfill, you should be able to focus the scrollable area.

Horizontally Scrollable Code Block

This code block is intentionally wide to demonstrate horizontal scrolling:

const myVeryLongFunctionNameThatExceedsTheContainerWidth = () => {
  console.log('This line is very long and will require horizontal scrolling to see it completely');
  return 'This demonstrates the scroll-focus polyfill making this element keyboard accessible';
};

How to Test

  1. Press Tab to navigate through the page
  2. When a code block is focused, you should see a focus outline
  3. Use arrow keys to scroll horizontally within the focused code block
  4. This works even in browsers that don't normally allow scrollable elements to receive focus
  5. Open the browser console to see debug logs (the polyfill is loaded with debug: true below)

Usage Examples

This demo uses the manual import with debug enabled. You can also use:

1. Auto-execute (simplest)

<!-- Just reference the package -->
<script src="https://unpkg.com/scroll-focus-polyfill"></script>

2. Force mode (always apply)

<!-- Always applies, skips detection -->
<script src="https://unpkg.com/scroll-focus-polyfill/force"></script>

3. With data attributes

<!-- Configure via data attributes -->
<script src="https://unpkg.com/scroll-focus-polyfill" 
        data-debug="true"
        data-selectors="pre, .scrollable"></script>

4. Manual control (like this demo)

<script type="module">
  import { applyPolyfill } from 'scroll-focus-polyfill/manual';
  applyPolyfill({ debug: true });
</script>
